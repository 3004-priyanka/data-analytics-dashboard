# 📊 Data Analytics Dashboard

This is a full-stack data analytics dashboard that helps visualize and manage data in a simple and interactive way. It was built to practice working with the MERN stack while creating a clean and responsive user interface.

## Features
- Interactive dashboard
- Data visualization
- Search and filtering
- Responsive design
- REST API integration

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Getting Started

```bash
# Frontend
cd frontend
npm install
npm start

# Backend
cd backend
npm install
npm run dev
```
